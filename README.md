## Installation
Install this globally from npm
```zsh
npm i -g katzu
```
or Clone this repository & install
```zsh
git clone https://github.com/katzEco/katzu.git
cd katzu
npm i -g .
```

## Usage
Just type `katzu` with my package name like 

**Basic Usage**
```zsh
katzu <package name>
```

**Example:**
```zsh
katzu tydon
```
> 1678518984243

or 
```zsh
katzu ranString
```
> wfnkot5cdi

## Options
- **help (--help)** - Show help
```zsh
tydon --help
```
- **version (--version)** - Show current version
```zsh
tydon --version
```
